# Code of Conduct

## Our Pledge

Everyone is welcome here. We are committed to providing a friendly, safe, and welcoming environment for all contributors, regardless of experience level, background, or identity.

## Our Standards

**Be respectful.** Treat others as you'd want to be treated. Disagreement is fine; disrespect is not.

**Be constructive.** Focus on the code, not the person. When reviewing PRs, explain _why_ something should change, not just that it should.

**Be helpful.** If someone asks a question, answer it or point them in the right direction. We were all beginners once.

## Scope

This code of conduct applies to all project spaces — issues, pull requests, discussions, and any other communication channel associated with Graph-Mem MCP.

## Enforcement

Instances of unacceptable behavior can be reported to the project maintainers by opening an issue or contacting the maintainers directly. All complaints will be reviewed.

Project maintainers who do not follow or enforce the Code of Conduct may be removed from the project team.
